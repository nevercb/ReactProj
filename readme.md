```
首页功能点分析：
1. 显示当前年月（下拉框点击月份可以切换）
2. 统计本月的收入和支出综合
3. 点击Tab切换不同视图（列表和图表）
4. 点击新建可以跳转到新建页面
5. 显示条目包括分裂图标，事件（主题），花费，日期，编辑和删除按钮
6. 点击编辑按钮会跳转到编辑页面（可与新建页共）
```
